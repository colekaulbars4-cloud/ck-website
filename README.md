# colekaulbars.com

Source for my personal website — built to showcase who I am and the projects
I've built while learning to create AI tools for community impact.

## What it is
A single-page portfolio site: who I am, the cause I care about, and links to
my builds — including a volunteer data cleaner, a SQL dashboard, a weather
event planner, an email automation, and a Claude-powered thank-you note
composer.

## How it's built
- Designed and built with Claude Design, exported as a single self-contained
  HTML file
- Hosted on Vercel with automatic redeploys whenever this repo updates
- Live at [colekaulbars.com](https://colekaulbars.com)

## Projects featured on the site
- [volunteer-data-cleaner](https://github.com/colekaulbars4-cloud/volunteer-data-cleaner) — Python data cleaning
- [volunteer-dashboard-sql](https://github.com/colekaulbars4-cloud/volunteer-dashboard-sql) — SQL queries + chart
